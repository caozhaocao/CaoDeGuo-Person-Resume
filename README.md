### 1. 智能物流   
时间：2018.06 — 2018.08  
语言：.Net  
项目描述：智能物流，为无人工厂也为自动化工厂 的一个材料运输环节，目前处在 半自动化工厂的协作方式，因为很多方面的不够完善，需要 人工进行协作，正在向 自动化工厂过渡。

已完成模块:  
1.在电镀上 同时容纳3个AGV车不让发生碰撞的交通规则，让AGV高效率的完成运输任务;  
2.AGV在低电压的情况下先去充电桩自动充电，然后接受运输任务。
